## moneyMade  
moneyMade는 웹으로 편리하게 소비/지출을 입력하고 나의 소비생활은 물론 
나와 비슷한 연령대의 소비트렌드 파악, 상품 최저가 검색, 금융 상품 추천까지 받을 수 있는 서비스입니다.
